# TwCounty

`geojson` of Taiwan County in 2010.

## Get data

- Get one data:
```
(your link)/twCounty/(Key)/
```

- Get all data:
```
(your link)/twCounty/
```

- For example, if you use `http://localhost:8000/twCounty/10018001/`, you will get geojson of `新竹市`.
